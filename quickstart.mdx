This quickstart walks you through running your first evaluation in Galtea using the Python SDK. By the end, you'll have evaluation results for your AI product.

<Steps>

  <Step title="Install SDK & Connect">Set up the Galtea Python SDK to interact with the platform.</Step>
  <Step title="Run Your Evaluation">Define your agent function and launch the evaluation with a single SDK call.</Step>

## 1. Tell Us About Your Product

If you're new to Galtea, start by creating an account and your first product in the dashboard: [Go to the dashboard](https://platform.galtea.ai/).

Follow the **Onboarding** checklist on the product page. It will prompt you for product details and guide you through generating specifications, tests, and metrics. These are all linked together — each specification defines what to test and how to score it.

## 2. Get Started with the SDK

You can use Galtea from the dashboard or via the Python SDK. This quickstart uses the Python SDK.

<Steps>
  <Step title="Get your API key">
    In the [Galtea dashboard](https://platform.galtea.ai/), copy the API key you created during your product page onboarding (or navigate to **[Settings](https://platform.galtea.ai/settings) > Generate API Key**).
  </Step>
  <Step title="Install the SDK">
    ```bash
    pip install galtea
    ```
  </Step>
  <Step title="Connect to the platform">
    ```python
    from galtea import Galtea

    # Verify the SDK is properly installed by initializing it
    # Note: Replace "YOUR_API_KEY" with your actual API key from the Galtea dashboard
    galtea = Galtea(api_key="YOUR_API_KEY")
    print("Galtea SDK installed successfully!")
    ```
  </Step>
</Steps>

## 3. Run Your Evaluation

Now connect Galtea to your product. You define an **agent function** — a wrapper that takes user input, calls your AI system, and returns its output. Then `evaluations.run()` handles everything: it resolves your specifications, iterates through test cases, runs your agent, and triggers evaluations using the linked metrics.

First, find your product and version IDs. You can copy them from the dashboard (click the **3-dot menu** on any item, then select **Copy ID**) or list them via the SDK:

```python
products = galtea.products.list()
for p in products:
    print(p.id, p.name)

versions = galtea.versions.list(product_id=products[0].id)
for v in versions:
    print(v.id, v.name)
```

Then set them:

```python
product_id = "your_product_id"
version_id = "your_version_id"
```

Next, define your agent. The SDK supports three agent function signatures — pick the one that matches how much context your agent needs:

<AgentCallables />

Finally, launch the evaluation:

```python
result = galtea.evaluations.run(
    version_id=version_id,
    agent=my_agent,
)

print(f"Evaluated {result['testCaseCount']} test cases across {len(result['specifications'])} specifications")
print(f"View results at: https://platform.galtea.ai/product/{product_id}")
```

That's it. The SDK resolves all specifications with linked metrics and tests, creates sessions for each test case, runs your agent, and submits evaluations — all in one call.

<Note>
  `evaluations.run()` handles both single-turn tests (RAG, security) and multi-turn conversational scenarios uniformly. No need to differentiate between test types in your code.
</Note>

## 4. See Your Results

Once your evaluations are submitted, open your product in the [Galtea platform](https://platform.galtea.ai/) and head to **Analytics**. Start with the aggregate scores to get a quick baseline, then slice by **version**, **test**, and **metric** to see where performance shifts.

The most useful part is drilling into individual test cases: you can inspect the exact prompt/output pair (or the full conversation for scenarios) and understand which specific inputs move the score up or down. This makes it easier to identify failure patterns, validate fixes in new versions, and iterate with confidence as you move toward production.

```python
print(f"View results at: https://platform.galtea.ai/product/{product_id}")
```
