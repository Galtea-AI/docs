---
title: "Link Metrics"
sidebarTitle: "Link Metrics"
description: "Link metrics to a user group."
---

## Example
```python
galtea.user_groups.link_metrics(
    user_group_id=user_group_id,
    metric_ids=[metric_id_1, metric_id_2],
)
```

## Parameters
<ResponseField name="user_group_id" type="string" required>
  The ID of the user group.
</ResponseField>
<ResponseField name="metric_ids" type="list[string]" required>
  List of metric IDs to link to the group.
</ResponseField>
