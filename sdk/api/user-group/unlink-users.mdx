---
title: "Unlink Users"
sidebarTitle: "Unlink Users"
description: "Remove users from a user group."
---

## Example
```python
galtea.user_groups.unlink_users(
    user_group_id=user_group_id,
    user_ids=[user_id_1],
)
```

## Parameters
<ResponseField name="user_group_id" type="string" required>
  The ID of the user group.
</ResponseField>
<ResponseField name="user_ids" type="list[string]" required>
  List of user IDs to remove from the group.
</ResponseField>
