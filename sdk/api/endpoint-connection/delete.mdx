---
title: 'Delete Endpoint Connection'
sidebarTitle: "Delete"
description: "Delete a specific endpoint connection by its ID."
---

## Example
```python
galtea.endpoint_connections.delete(
    endpoint_connection_id=endpoint_connection_id
)
```

## Parameters
<ResponseField name="endpoint_connection_id" type="string" required>
  The ID of the endpoint connection to delete.
</ResponseField>
