## Returns
Returns a [Trace](/concepts/product/version/session/trace) object.

## Example
```python
trace = galtea.traces.get(trace_id=trace_id)
```

## Parameters
<ResponseField name="trace_id" type="string" required>
  The unique identifier of the trace to retrieve.
</ResponseField>
