---
title: 'Start Collection Context'
sidebarTitle: "Start Collection Context"
description: "Initialize trace collection for the current context."
---

## Returns
This method returns `None`.

## Example
```python
galtea.traces.start_collection_context()
```

## Parameters
This method takes no parameters.

<Note>
  If you're using `galtea.inference_results.generate()`, this is called automatically. You only need to call this method when using manual trace collection.
</Note>
