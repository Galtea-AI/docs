title: 'Deleting Test Case'
sidebarTitle: "Delete"
description: "Delete a specific test case by its ID."
---

## Example
```python
galtea.test_cases.delete(test_case_id=test_case_id)
```

## Parameters
<ResponseField name="test_case_id" type="string" required>
  The ID of the test case you want to delete.
</ResponseField>
