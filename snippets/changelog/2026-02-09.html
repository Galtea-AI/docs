Enhanced Simulation and UI Improvements

2026-02-09

SDK & Simulation Upgrades

We have significantly improved the flexibility of the Galtea SDK to better fit your existing workflows.

  • Unified Simulation Method: The simulator.simulate method has been upgraded to handle both single-turn and multi-turn simulations.
  • Simplified Agent Integration: The agent property now accepts direct methods with various signatures, removing the need for wrapper classes.

New Platform Features

AI-Powered Product Definition

Bootstrapping a new product context is now faster than ever. You can upload a set of files (documentation, knowledge base, etc.), and Galtea will use AI to automatically generate a comprehensive definition for you.

Advanced Endpoint Connections

You can now create endpoint connections with distinct configurations for the Initialization, the Conversation, and the Finalization of the conversation.

Reliability and UX Improvements

  • Graceful Metric Handling: Incompatible metrics now result in a "skipped" status rather than throwing an error.
  • Transition to Custom Metrics: We have deprecated DeepEval metrics in favor of our own custom implementation to enhance reliability.
  • Better Test Creation: The Test creation form has been polished for improved usability.
  • Stability: General improvements to platform reliability.
